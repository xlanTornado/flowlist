# 待办清单桌面应用 — 产品计划书（Windows 版）

## 一、产品定位

| 维度 | 描述 |
|------|------|
| **产品名** | FlowList（暂定） |
| **一句话** | 轻量、离线、高颜值的 Windows 待办清单工具 |
| **目标用户** | 20-35 岁年轻群体，学生、上班族、效率爱好者 |
| **核心卖点** | 单文件便携运行/绿色免安装、现代清新 UI、功能完整 |

## 二、技术架构

| 层面 | 选型 | 理由 |
|------|------|------|
| 桌面框架 | **Tauri v2** | 打包后 ~5-8MB 单 exe，用户无需装任何运行时 |
| 前端 | React 18 + TypeScript | 组件化、类型安全 |
| 样式 | Tailwind CSS | 快速实现清新现代 UI |
| 本地存储 | SQLite | 嵌入式数据库，零配置 |
| 状态管理 | Zustand | 轻量够用 |
| 系统通知 | Windows Toast Notification | 原生通知体验 |
| 全局快捷键 | Tauri global shortcut plugin | 后台监听 |
| 开机自启 | Tauri autostart plugin | 注册到 Windows 启动项 |
| 打包 | **MSI 安装包 + 便携版 exe** | 兼顾两种分发场景 |

## 三、功能规划

### V1.0 — 核心闭环

| 模块 | 功能 |
|------|------|
| 任务管理 | 新建/编辑/删除任务，标题+备注，子任务（两级） |
| 清单分类 | 多清单管理，颜色标记区分 |
| 优先级 | P1(红) / P2(橙) / P3(蓝) / P4(灰) |
| 截止日期 | 日期+时间选择，过期任务红色警示 |
| 提醒通知 | Windows 原生 Toast 通知，到期前 15min/1h/1d |
| 重复任务 | 每日/每周/每月/工作日/自定义 |
| 标签 | 自定义标签，多标签，按标签筛选 |
| 搜索 | 全文模糊搜索 |
| 排序 | 按优先级/日期/创建时间/字母排序 |

### V1.1 — 效率增强

| 功能 | 说明 |
|------|------|
| 全局快捷键 | `Ctrl+Shift+N` 快速新建、`Ctrl+Shift+F` 全局搜索（驻留托盘时可用） |
| 拖拽排序 | 任务和清单均支持拖拽 |
| 批量操作 | 多选后批量完成/删除/移动 |
| 今日视图 | 聚合今天到期的所有任务 |
| 已完成归档 | 自动归档，可按日期回溯 |
| 系统托盘 | 最小化到托盘，托盘图标显示今日未完成任务数 |

### V1.2 — 数据洞察

| 功能 | 说明 |
|------|------|
| 统计面板 | 完成率趋势图、周热力图、标签分布 |
| 数据导出 | JSON/CSV 导出 |
| 自动备份 | 本地定时备份 SQLite 文件 |

## 四、数据模型

```
lists       — id, name, color, sort_order, created_at
tasks       — id, list_id, parent_id, title, note, priority, due_date,
              is_completed, completed_at, sort_order, repeat_rule, created_at, updated_at
tags        — id, name, color
task_tags   — task_id, tag_id
reminders   — id, task_id, remind_at, triggered
settings    — key, value
```

## 五、UI 设计方向

- **布局**：三栏 — 左侧清单列表 | 中间任务列表 | 右侧详情面板
- **配色**：浅色 + 深色双主题，马卡龙色系清单颜色
- **字体**：Segoe UI（Windows 11 原生，清晰现代）
- **窗口**：支持 Acrylic/Mica 毛玻璃效果（Windows 11），圆角窗口
- **交互**：右键菜单、键盘全操作、流畅过渡动画

## 六、Windows 特有体验

| 特性 | 实现 |
|------|------|
| 系统托盘 | 最小化到托盘，红点显示今日未完成数 |
| 原生通知 | Windows Toast Notification，点击通知直接打开对应任务 |
| 开机自启 | 注册到 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` |
| 任务栏角标 | 显示未完成任务计数 |
| 跳转列表 | 右键任务栏图标快速新建任务 |
| 便携版 | 绿色免安装 zip 包，解压即用 |

## 七、开发 Roadmap

| 阶段 | 内容 |
|------|------|
| **Phase 0**（第 1 周） | Tauri 脚手架 + React/Tailwind + SQLite 连接 + 项目结构 |
| **Phase 1**（第 2-3 周） | 任务 CRUD、清单管理、优先级、三栏布局 UI |
| **Phase 2**（第 4-5 周） | 截止日期、提醒通知、子任务、标签、搜索 |
| **Phase 3**（第 6 周） | 重复任务、拖拽排序、系统托盘、全局快捷键 |
| **Phase 4**（第 7 周） | 统计面板、导入导出、深色模式、Mica 效果 |
| **Phase 5**（第 8 周） | 测试、打包 MSI + 便携版、发布 |

## 八、打包与分发

| 格式 | 说明 |
|------|------|
| `.msi` 安装包 | 标准 Windows 安装体验，支持卸载 |
| `.exe` 便携版 | 绿色免安装，U 盘携带即用 |
| 自动更新 | Tauri updater plugin，检测 GitHub Release 新版本 |
